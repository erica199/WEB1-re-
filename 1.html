HTML

HyperText Markup Language(HTML) is the standard Markup language for creation web pages and web applications.

수업한 내용정리:
**web을 지배하는 중요한 문법: TAG**
1. strong: 글자를 굵게(진하게) 표시하기
2. u(underline): 밑줄 표시
3. h1,h2,h3,h4,h5,h6: 제목 표시(두껍고, 줄바꿈 적용,글자가 커질수록 글씨는 작아짐)
4. br: 줄바꿈, h1을 제외하면 여태까지 배운 태그들은 줄바꿈이 자동적으로 적용되지 않음, 여태까지 배운 다른 태그들과 달리 닫을 필요가 없다
5. p: 문단 설정하기, 여백이 정해져 있음
